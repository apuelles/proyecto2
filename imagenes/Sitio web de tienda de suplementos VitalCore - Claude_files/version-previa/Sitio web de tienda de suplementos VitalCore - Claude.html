  • Sitio web de tienda de suplementos VitalCore

claude neceesito ayuda para programar un sitio web. El sitio web sera de una empresa llamada VitalCore que vende 4 suplementos para el gimnasio: creatina, proteina, electrolitos y pre workout. El codigo debe ser HTML, CSS y javascript. La pagina web debe tener un carrito que cada vez que agregas un producto se va sumando (si no tenes nada en el carrito al lado aparecera 0, si pones un producto 1, si pones dos 2, etc). Tambien debe tener una seccion de "finalizar compra" si tocas el carrito, pero no hace falta que sea totalmente funcional esa parte. Estaba pensando que la mejor paleta de colores seria una escala de grises, blancos o negros.

¡Acá tenés el sitio de VitalCore listo! Esto es lo que incluye:

Estructura del sitio:

  • Navbar fija con logo, links de navegación y el botón de carrito con contador
  • Hero section con título grande y llamado a la acción
  • Sección de productos (Creatina, Proteína, Electrolitos, Pre Workout) con precio, descripción y botón de agregar

  • Footer

Funcionalidades del carrito:

  • El contador arriba empieza en 0 y suma/resta en tiempo real con una animación de rebote
  • Al hacer clic en "Carrito" se abre un panel lateral con los ítems, podés cambiar cantidades o eliminar productos
  • Hay un botón "Finalizar Compra" que abre un modal con un formulario de datos y resumen del pedido
  • Al confirmar, aparece un mensaje de éxito y el carrito se vacía
  • Toasts de notificación al agregar productos

Diseño:

  • Paleta en escala de grises sobre fondo negro muy oscuro
  • Tipografías contrastantes: Bebas Neue para los títulos y Courier Prime para los detalles técnicos
  • Efecto de ruido de fondo, grilla decorativa y animaciones de entrada al hacer scroll
